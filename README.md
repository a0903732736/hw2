# 題目一：RC 充電電路

## 1. 工程問題
電容在接上直流電源後，電壓如何隨時間變化？

---

## 2. 建模（KVL）
根據電壓迴路定律：

V = IR + Vc

又因為：
I = C dVc/dt

可得微分方程：

RC dVc/dt + Vc = E

---

## 3. 解微分方程
整理為：

dVc/dt = (E - Vc) / RC

分離變數：

∫1/(E - Vc) dVc = ∫1/(RC) dt

積分後：

-ln|E - Vc| = t/RC + C

整理可得：

Vc(t) = E - A e^(-t/RC)

---

## 4. 套用初始條件
若 t = 0 時，Vc(0) = 0：

A = E

---

## 5. 最終解
Vc(t) = E(1 - e^(-t/RC))

---

## 6. 工程意義
- RC = 時間常數 τ
- t = τ 時，電壓約為 63%
- t ≈ 5τ 時，電壓接近穩定
- 最終 Vc → E

---

## 7. 應用
- 定時器
- 濾波器
- 開關去彈跳
